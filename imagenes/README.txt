CARPETA DE IMÁGENES — Espacio Antü Ayün
========================================

El sitio actual NO requiere imágenes para funcionar, ya que todo el
diseño visual se construyó con SVG, CSS y gradientes (íconos vectoriales,
formas decorativas, colores de marca).

Sin embargo, esta carpeta queda lista para que puedas agregar imágenes
en el futuro si lo deseas. Sugerencias de imágenes recomendadas en
formato .png:

1. logo.png
   - Logo oficial de Espacio Antü Ayün.
   - Uso sugerido: reemplazar el logo de texto en el <nav> del index.html.
   - Tamaño recomendado: 200x60 px aprox.

2. hero-equipo.png
   - Foto del equipo profesional o de las instalaciones.
   - Uso sugerido: reemplazar las tarjetas decorativas del hero.
   - Tamaño recomendado: 800x900 px aprox.

3. fachada-centro.png
   - Foto de la fachada o entrada del BCenter Consistorial.
   - Uso sugerido: sección de contacto o "nosotros".
   - Tamaño recomendado: 1200x800 px aprox.

4. equipo-1.png, equipo-2.png, equipo-3.png, equipo-4.png, equipo-5.png
   - Fotos de cada profesional del equipo.
   - Uso sugerido: agregar una nueva sección "Nuestro equipo".
   - Tamaño recomendado: 600x600 px (cuadradas).

5. og-image.png
   - Imagen para compartir en redes sociales (Open Graph).
   - Uso sugerido: agregar como meta tag en el <head>.
   - Tamaño recomendado: 1200x630 px.

CÓMO REFERENCIAR LAS IMÁGENES EN EL HTML:
------------------------------------------
Una vez agregadas, se llaman con ruta relativa así:

  <img src="imagenes/logo.png" alt="Logo Espacio Antü Ayün">
  <img src="imagenes/hero-equipo.png" alt="Equipo profesional">

NOTA: Si decides agregar imágenes, recuerda optimizarlas antes (TinyPNG,
Squoosh) para que el sitio cargue rápido.
